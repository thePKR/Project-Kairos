# Contributing to Project Kairos 🏭🧠

We love your input! We want to make contributing to this autonomous software factory as transparent and easy as possible. 

## The Local Development Loop

1. **Fork the repo** and create your branch from `main`.
2. Ensure you have added the required `NVIDIA_API_KEY` to your local `.env` so the swarm can actually think. (Get a free key from build.nvidia.com)
3. If you integrate new Langchain Nodes to `src/main.py`, ensure the swarm still successfully triggers the Human-in-the-Loop Huddle.
4. Issue that pull request!

## Pull Request Process

1. Ensure the Github Action (Python CI) passes and the linter `flake8` throws no hard syntax errors.
2. Update the `README.md` with details of changes to the interface, if applicable.
3. You may merge the Pull Request once you have the sign-off of at least one other developer (or the Swarm itself).

## Any questions?
Create an Issue! We utilize strict Issue Templates for Bug Reports and Feature Requests.
